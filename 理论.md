   模块1：脉冲神经元（SNN）——AI的“神经元”
   核心思想：传统ANN的神经元输出连续的浮点数，而生物神经元输出的是离散的“脉冲”——只有当膜电位超过阈值时才发射，平时保持沉默。这是大脑超高能效的根本原因。

生物对应：大脑中的神经元（≈860亿个）通过电化学脉冲通信，平均放电率仅1-10 Hz，但能完成极其复杂的认知任务。

原理：Leaky Integrate-and-Fire（LIF）模型
LIF是描述神经元膜电位动态的最简模型：

text
τ \* dV/dt = -(V - V\_rest) + R \* I(t)

当 V >= V\_threshold 时:
发射脉冲（spike = 1）
V ← V\_reset
物理意义：

V：神经元膜电位（类比电容电压）

τ：膜时间常数，控制电位衰减速度（类比RC电路的时间常数）

I(t)：来自其他神经元的输入电流

V\_threshold：放电阈值（约-55mV）

V\_reset：放电后复位电位（约-70mV）

"""
模块1：脉冲神经元（LIF模型）
可以直接复制到Python文件中运行
"""

import numpy as np
import matplotlib.pyplot as plt

class LIFNeuron:
"""Leaky Integrate-and-Fire 神经元"""

```
def __init__(self, tau=10.0, threshold=1.0, v_reset=0.0, v_rest=0.0):
    """
    Args:
        tau: 膜时间常数 (ms)
        threshold: 放电阈值
        v_reset: 复位电位
        v_rest: 静息电位
    """
    self.tau = tau
    self.threshold = threshold
    self.v_reset = v_reset
    self.v_rest = v_rest
    self.v = v_rest  # 当前膜电位
    
def step(self, current, dt=1.0):
    """
    单步仿真
    
    Args:
        current: 输入电流
        dt: 时间步长 (ms)
    
    Returns:
        spike: 是否发射脉冲 (0或1)
    """
    # LIF微分方程：dV/dt = -(V - V_rest)/tau + current
    dv = (-(self.v - self.v_rest) + current) * (dt / self.tau)
    self.v += dv
    
    # 检查是否超过阈值
    if self.v >= self.threshold:
        self.v = self.v_reset
        return 1
    return 0

def reset(self):
    self.v = self.v_rest
```

模块2：赫布可塑性（STDP）——AI的“学习法则”
核心思想：传统深度学习使用全局的反向传播（Backprop），而大脑使用的是局部学习规则——突触强度的改变只取决于前后神经元的活动。这被总结为赫布定律：“一起激发的神经元，连接在一起”（Cells that fire together, wire together）。

生物对应：突触可塑性是学习和记忆的细胞基础。STDP（Spike-Timing-Dependent Plasticity）是赫布定律的精确时间版本：如果突触前神经元先于突触后神经元放电（<20ms），突触增强（LTP）；反之则减弱（LTD）。

原理：STDP的数学公式
经典的STDP更新规则（Song et al., 2000）：

text
Δw = A\_plus \* exp(-Δt/τ\_plus)  if Δt > 0  (pre 先于 post)
Δw = -A\_minus \* exp(Δt/τ\_minus) if Δt < 0 (post 先于 pre)

其中 Δt = t\_post - t\_pre
A\_plus, A\_minus：增强/减弱的幅度（通常 A\_minus > A\_plus，体现“减弱强于增强”的生物特性）

τ\_plus, τ\_minus：时间窗口常数（≈20ms）

"""
模块2：赫布可塑性（STDP）
直接复制运行，演示突触权重如何根据脉冲时序自动调整
"""

import numpy as np
import matplotlib.pyplot as plt
from collections import deque

class STDPSynapse:
"""STDP学习规则的突触"""

```
def __init__(self, w_init=0.5, A_plus=0.01, A_minus=0.012, 
             tau_plus=20.0, tau_minus=20.0, w_min=0.0, w_max=1.0):
    """
    Args:
        w_init: 初始权重
        A_plus: 增强幅度
        A_minus: 减弱幅度（通常 > A_plus）
        tau_plus: 增强时间窗口 (ms)
        tau_minus: 减弱时间窗口 (ms)
        w_min, w_max: 权重边界
    """
    self.w = w_init
    self.A_plus = A_plus
    self.A_minus = A_minus
    self.tau_plus = tau_plus
    self.tau_minus = tau_minus
    self.w_min = w_min
    self.w_max = w_max
    
    # 记录最近脉冲时间
    self.pre_spike_time = None
    self.post_spike_time = None

def update(self, pre_spike=False, post_spike=False, t=0):
    """
    根据前后脉冲更新权重
    
    Args:
        pre_spike: 突触前神经元是否刚发射
        post_spike: 突触后神经元是否刚发射
        t: 当前时间 (ms)
    """
    # 前后都发射？记录时间并计算Δw
    if pre_spike:
        self.pre_spike_time = t
        if self.post_spike_time is not None:
            dt = self.post_spike_time - t  # post - pre
            if dt > 0:  # post 在 pre 之后 → LTP（增强）
                dw = self.A_plus * np.exp(-dt / self.tau_plus)
            else:       # post 在 pre 之前 → LTD（减弱）
                dw = -self.A_minus * np.exp(dt / self.tau_minus)
            self.w += dw
            self.w = np.clip(self.w, self.w_min, self.w_max)
            return dw
    
    if post_spike:
        self.post_spike_time = t
        if self.pre_spike_time is not None:
            dt = t - self.pre_spike_time  # post - pre
            if dt > 0:  # post 在 pre 之后 → LTP
                dw = self.A_plus * np.exp(-dt / self.tau_plus)
            else:       # post 在 pre 之前 → LTD
                dw = -self.A_minus * np.exp(dt / self.tau_minus)
            self.w += dw
            self.w = np.clip(self.w, self.w_min, self.w_max)
            return dw
    
    return 0.0
```

模块3：预测编码（PCN）——AI的“世界模型”
核心思想：大脑不是被动接收信息，而是不断预测下一秒会发生什么，只处理预测误差。这极大压缩了信息量，也是我们拥有“常识”和“物理直觉”的原因。

生物对应：新皮层具有层次化结构，高层区域向低层发送“预测”，低层向上发送“预测误差”。这解释了为什么我们能闭眼想象场景、预测球的轨迹。

原理：层次化预测编码的数学框架
预测编码的核心是自由能原理（Free Energy Principle）：任何自组织系统必须最小化其变分自由能（即“意外感”）。对于神经网络，自由能可分解为各层的预测误差：

text
F = Σ\_l ||x\_l - f\_l(x\_{l+1})||² / (2σ\_l²)

其中：

- x\_l：第l层的真实状态（从下向上的感知信号）
- f\_l(x\_{l+1})：从高层向下传递的预测
  推理（感知）：固定权重，迭代更新各层状态以最小化预测误差
  学习（记忆）：固定状态，更新权重（突触）以更好地预测

可运行代码
python
"""
模块3：预测编码网络（Predictive Coding Network）
演示一个3层PCN如何学会预测简单的时序模式
"""

import numpy as np
import matplotlib.pyplot as plt

class PredictiveCodingLayer:
"""单层预测编码"""

```
def __init__(self, input_size, hidden_size, learning_rate=0.01):
    """
    Args:
        input_size: 输入维度
        hidden_size: 隐藏状态维度
        learning_rate: 学习率
    """
    self.input_size = input_size
    self.hidden_size = hidden_size
    
    # 权重：从高层到底层的预测映射
    self.W = np.random.randn(input_size, hidden_size) * 0.1
    
    # 状态变量（将被推理过程优化）
    self.x = np.zeros(hidden_size)
    
    self.lr = learning_rate

def predict(self, x_above):
    """从高层生成预测"""
    return self.W @ x_above

def inference_step(self, prediction_from_above, error_from_below, inference_lr=0.05):
    """
    单步推理：更新本层状态以最小化预测误差
    
    Args:
        prediction_from_above: 来自高层的预测
        error_from_below: 来自低层的预测误差
        inference_lr: 推理学习率
    """
    # 梯度 = -(高层预测误差 + 本层预测误差)
    # 自由能F = ||x - f(x_above)||^2 / (2*sigma^2) + ...
    # dF/dx = -(prediction_from_above - x) - W.T @ error_from_below
    # 梯度下降: x -= lr * dF/dx = x + lr * (prediction_from_above - x + W.T @ error_from_below)
    grad = (prediction_from_above - self.x) + self.W.T @ error_from_below
    self.x += inference_lr * grad
    return self.x

def learning_step(self, error, x_above):
    """
    更新权重（学习）
    
    Args:
        error: 预测误差
        x_above: 高层的状态
    """
    # 自由能对W的梯度: dF/dW = -sigma * outer(error, x_above)
    # 梯度下降: W -= lr * dF/dW = W + lr * sigma * outer(error, x_above)
    sigma = 1.0  # 精度
    dw = sigma * np.outer(error, x_above)
    self.W += self.lr * dw
```

class PredictiveCodingNetwork:
"""三层预测编码网络"""

```
def __init__(self, sizes, learning_rate=0.01):
    """
    Args:
        sizes: [input_size, hidden1_size, hidden2_size]
        learning_rate: 权重学习率
    """
    self.layers = []
    self.sizes = sizes
    
    for i in range(len(sizes) - 1):
        self.layers.append(
            PredictiveCodingLayer(sizes[i], sizes[i+1], learning_rate)
        )
    
    # 输入层状态（固定为观测值）
    self.input_state = np.zeros(sizes[0])

def infer(self, observation, n_steps=20, inference_lr=0.05):
    """
    推理：给定观测，迭代更新所有隐藏层状态
    
    Args:
        observation: 当前观测
        n_steps: 推理迭代次数
        inference_lr: 推理学习率
    """
    self.input_state = observation.copy()
    
    for _ in range(n_steps):
        # 自顶向下传播预测
        # 顶层
        pred_top = self.layers[1].predict(self.layers[1].x)
        
        # 中间层
        error_bottom = self.input_state - self.layers[0].predict(self.layers[0].x)
        self.layers[0].inference_step(pred_top, error_bottom, inference_lr)
        
        # 顶层
        error_mid = self.layers[0].x - pred_top
        self.layers[1].inference_step(np.zeros(self.sizes[2]), error_mid, inference_lr)

def learn(self, n_steps=20):
    """学习：根据当前状态更新所有权重"""
    # 计算各层误差
    error_0 = self.input_state - self.layers[0].predict(self.layers[0].x)
    error_1 = self.layers[0].x - self.layers[1].predict(self.layers[1].x)
    
    # 更新权重
    self.layers[0].learning_step(error_0, self.layers[0].x)
    self.layers[1].learning_step(error_1, self.layers[1].x)

def predict_next(self):
    """预测下一个状态"""
    return self.layers[0].predict(self.layers[0].x)
```

模块4：主动推理（Active Inference）——AI的“好奇心引擎”
核心思想：AI不应该被动等待数据，而应该主动选择行动来减少对世界的“意外感”。这就是内在动机（好奇心）的数学形式——选择能最大化信息增益的行动。

生物对应：婴儿会主动抓取、扔东西、探索环境，这不是因为有人给奖励，而是因为大脑天生追求“减少不确定性”。前额叶和基底节共同实现了这种主动推理。

原理：自由能与期望自由能
主动推理的核心方程：

text
G(π) = E\_Q\[log Q(s|π) - log P(o,s|π)]
\= - E\_Q\[log P(o|s)] + D\_KL\[Q(s|π) || P(s|π)]
\_\_\_\_\_\_ epistemic \____/   \____ pragmatic \_\_\_\_\_\_/
认知价值（Epistemic Value）：选择行动后，预期能学到多少新信息（减少多少不确定性）

实用价值（Pragmatic Value）：选择行动后，预期能获得多少奖励

婴儿期以认知价值为主导（纯粹的好奇心），成年后实用价值逐渐增加。

"""
模块4：主动推理（Active Inference）
演示AI如何基于"好奇心"主动选择行动来探索未知环境
"""

import numpy as np
import matplotlib.pyplot as plt
from scipy.special import softmax

class ActiveInferenceAgent:
"""主动推理智能体"""

```
def __init__(self, n_states=5, n_actions=3, temperature=1.0):
    """
    Args:
        n_states: 环境状态数量
        n_actions: 可选行动数量
        temperature: 探索温度（越高越随机）
    """
    self.n_states = n_states
    self.n_actions = n_actions
    self.temperature = temperature
    
    # 先验信念（对环境状态的不确定性）
    self.belief = np.ones(n_states) / n_states
    
    # 转移模型：P(s'|s,a) —— 智能体对"行动会如何改变状态"的信念
    self.transition_model = np.ones((n_states, n_actions, n_states)) / n_states
    
    # 观测模型：P(o|s) —— 智能体对"状态会产生什么观测"的信念
    self.observation_model = np.eye(n_states) + 0.1
    self.observation_model /= self.observation_model.sum(axis=1, keepdims=True)
    
    # 偏好（先验目标）
    self.preferences = np.ones(n_states) / n_states
    
    # 学习历史
    self.history = []

def perceive(self, observation):
    """感知：根据观测更新信念（贝叶斯推断）"""
    # P(s|o) ∝ P(o|s) P(s)
    likelihood = self.observation_model[:, observation]
    self.belief = likelihood * self.belief
    self.belief /= self.belief.sum()
    return self.belief

def expected_free_energy(self, action):
    """
    计算选择某个行动的期望自由能
    （负值代表"好的"行动——能减少不确定性和/或获得奖励）
    """
    # 预测下一个状态分布
    predicted_next_belief = self.transition_model[:, action, :].T @ self.belief
    
    # 1. 认知价值（信息增益）
    # 使用互信息衡量：I(o;s) = H[P(o)] - E_s[H[P(o|s)]]
    # 这比简单的熵差更精确，因为它考虑了观测对状态的信息量
    marginal_obs = self.observation_model.T @ predicted_next_belief
    H_marginal = -np.sum(marginal_obs * np.log(marginal_obs + 1e-10))
    H_conditional = sum(predicted_next_belief[s] * 
                       (-np.sum(self.observation_model[s] * np.log(self.observation_model[s] + 1e-10)))
                       for s in range(self.n_states))
    epistemic_value = max(0, H_marginal - H_conditional)
    
    # 2. 实用价值（与偏好的对齐程度）
    pragmatic_value = np.sum(predicted_next_belief * np.log(self.preferences + 1e-10))
    
    # 期望自由能 = -认知价值 - 实用价值（我们要最小化这个）
    # 转换为"质量分数"（越高越好）
    quality = epistemic_value + pragmatic_value
    
    return quality, epistemic_value, pragmatic_value

def select_action(self):
    """选择行动：软最大化（探索-利用权衡）"""
    qualities = []
    for a in range(self.n_actions):
        q, ep, pr = self.expected_free_energy(a)
        qualities.append(q)
    
    qualities = np.array(qualities)
    
    # 软最大化（温度控制随机性）
    probs = softmax(qualities / self.temperature)
    action = np.random.choice(self.n_actions, p=probs)
    
    return action, qualities, probs

def learn(self, action, next_state):
    """学习：更新转移模型"""
    # 根据经验更新转移概率
    self.transition_model[:, action, next_state] += 0.1
    self.transition_model[:, action, :] /= self.transition_model[:, action, :].sum(axis=1, keepdims=True)

def step(self, observation):
    """单步决策循环"""
    # 1. 感知
    self.perceive(observation)
    
    # 2. 选择行动
    action, qualities, probs = self.select_action()
    
    # 记录
    self.history.append({
        'belief': self.belief.copy(),
        'action': action,
        'qualities': qualities.copy()
    })
    
    return action
```

模块5：超维计算（HDC/VSA）——AI的“记忆系统”
核心思想：大脑如何存储和检索海量的概念、情景和关系？超维计算（Hyperdimensional Computing）提供了一种数学优雅的方案：用超高维度（>10,000维）的随机向量来表示信息，利用“高维空间的近乎正交性”实现联想记忆。

原理：HDC的三种基本运算

1. 捆绑（Bundling）：A ⊕ B = A + B
   → 表示集合 {"苹果", "香蕉"}
2. 绑定（Binding）：A ⊗ B = A XOR B（或循环卷积）
   → 表示键值对 "颜色" ⊗ "红色"
3. 排列（Permutation）：ρ(A)
   → 表示序列 \["A", "B", "C"]
   关键性质：

向量相似度用余弦距离衡量

相似向量代表相似概念

运算保持维度不变

"""
模块5：超维计算（HDC）——记忆系统
演示如何用高维向量表示、存储和检索概念
"""

import numpy as np
import matplotlib.pyplot as plt
from collections import defaultdict

class HDMemory:
"""超维记忆系统"""

```
def __init__(self, dim=10000, seed=42):
    """
    Args:
        dim: 向量维度（越高越好，典型值10000）
        seed: 随机种子
    """
    self.dim = dim
    np.random.seed(seed)
    
    # 存储的概念向量
    self.concepts = {}
    
    # 联想记忆（键值对）
    self.associative_memory = {}
    
    # 情景记忆（序列）
    self.episodic_memory = []

def random_vector(self):
    """生成随机高维向量（±1）"""
    return np.random.choice([-1, 1], size=self.dim)

def cosine_similarity(self, a, b):
    """余弦相似度"""
    return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))

def bundle(self, *vectors):
    """
    捆绑操作（⊕）：多向量叠加
    表示集合或类别
    """
    result = np.zeros(self.dim)
    for v in vectors:
        result += v
    # 二值化（保持±1）
    return np.sign(result)

def bind(self, a, b):
    """
    绑定操作（⊗）：逐元素相乘
    表示键值对
    """
    return a * b

def permute(self, v, shift=1):
    """
    排列操作（ρ）：循环移位
    表示序列位置
    """
    return np.roll(v, shift)

def add_concept(self, name, vector=None):
    """添加新概念"""
    if vector is None:
        vector = self.random_vector()
    self.concepts[name] = vector
    return vector

def store_association(self, key, value):
    """存储联想：key → value"""
    if key not in self.concepts:
        self.add_concept(key)
    if value not in self.concepts:
        self.add_concept(value)
    
    # 用绑定存储键值对
    association = self.bind(self.concepts[key], self.concepts[value])
    self.associative_memory[key] = value
    return association

def retrieve(self, query_vector, top_k=1):
    """根据查询向量检索最相似的概念"""
    similarities = []
    for name, vec in self.concepts.items():
        sim = self.cosine_similarity(query_vector, vec)
        similarities.append((name, sim))
    
    similarities.sort(key=lambda x: x[1], reverse=True)
    return similarities[:top_k]

def store_episode(self, sequence):
    """存储情景（序列）"""
    # 用排列编码序列
    episode_vector = np.zeros(self.dim)
    for i, item in enumerate(sequence):
        if item not in self.concepts:
            self.add_concept(item)
        # 位置编码
        position_encoded = self.permute(self.concepts[item], i)
        episode_vector += position_encoded
    
    self.episodic_memory.append({
        'sequence': sequence,
        'vector': np.sign(episode_vector)
    })
    
    return np.sign(episode_vector)

def query_episode(self, partial_sequence):
    """用部分序列查询完整情景"""
    # 编码部分序列
    query_vec = np.zeros(self.dim)
    for i, item in enumerate(partial_sequence):
        if item in self.concepts:
            query_vec += self.permute(self.concepts[item], i)
    
    query_vec = np.sign(query_vec)
    
    # 找最相似的情景
    best_match = None
    best_sim = -1
    
    for ep in self.episodic_memory:
        sim = self.cosine_similarity(query_vec, ep['vector'])
        if sim > best_sim:
            best_sim = sim
            best_match = ep
    
    return best_match, best_sim
```

模块6：世界模型——AI的“想象力引擎”
核心思想：真正的智能需要能够在“脑海中”模拟未来——这就是世界模型。给定当前状态和可能的行动，世界模型预测下一个状态。这让AI能够规划，而不仅仅是反应。

生物对应：前额叶皮层和海马体共同实现了“心理模拟”——我们可以想象明天去超市的路线，而不需要真的走一遍。

原理：循环状态空间模型（RSSM）
世界模型通常由以下组件构成：

编码器：观测 -> 潜在状态

动态模型：潜在状态 + 行动 -> 下一潜在状态

解码器：潜在状态 -> 重建观测

奖励模型：潜在状态 -> 预期奖励

可运行代码
python
"""
模块6：世界模型（World Model）
演示AI如何学习环境的内部模型并进行"想象"
"""

import numpy as np
import matplotlib.pyplot as plt

class WorldModel:
"""简单的循环世界模型"""

```
def __init__(self, obs_dim=2, state_dim=8, action_dim=2, lr=0.01):
    """
    Args:
        obs_dim: 观测维度
        state_dim: 潜在状态维度
        action_dim: 行动维度
        lr: 学习率
    """
    self.obs_dim = obs_dim
    self.state_dim = state_dim
    self.action_dim = action_dim
    self.lr = lr
    
    # 编码器：观测 → 状态（均值和对数方差）
    self.encoder_W = np.random.randn(state_dim, obs_dim) * 0.1
    self.encoder_b = np.zeros(state_dim)
    
    # 动态模型：状态 + 行动 → 下一状态
    self.dynamics_W = np.random.randn(state_dim, state_dim + action_dim) * 0.1
    self.dynamics_b = np.zeros(state_dim)
    
    # 解码器：状态 → 观测
    self.decoder_W = np.random.randn(obs_dim, state_dim) * 0.1
    self.decoder_b = np.zeros(obs_dim)
    
    # 训练历史
    self.loss_history = []

def encode(self, obs):
    """编码观测为潜在状态"""
    return np.tanh(self.encoder_W @ obs + self.encoder_b)

def predict_next_state(self, state, action):
    """预测下一个状态"""
    # 拼接状态和行动
    sa = np.concatenate([state, action])
    next_state = np.tanh(self.dynamics_W @ sa + self.dynamics_b)
    return next_state

def decode(self, state):
    """解码状态为观测"""
    return self.decoder_W @ state + self.decoder_b

def imagine_trajectory(self, start_obs, action_sequence):
    """
    想象：给定起始观测和行动序列，模拟未来轨迹
    
    Args:
        start_obs: 起始观测
        action_sequence: 行动序列列表
    
    Returns:
        imagined_states: 想象的状态序列
        imagined_obs: 想象的观测序列
    """
    state = self.encode(start_obs)
    
    states = [state]
    obs = [start_obs]
    
    for action in action_sequence:
        state = self.predict_next_state(state, action)
        next_obs = self.decode(state)
        
        states.append(state)
        obs.append(next_obs)
    
    return np.array(states), np.array(obs)

def train_step(self, obs, action, next_obs):
    """单步训练（监督学习）"""
    # 前向传播
    state = self.encode(obs)
    pred_next_state = self.predict_next_state(state, action)
    pred_next_obs = self.decode(pred_next_state)
    
    # 损失：预测观测 vs 真实观测
    loss = np.mean((pred_next_obs - next_obs) ** 2)
    
    # 简化梯度更新（数值梯度）
    eps = 1e-5
    grad_W = np.zeros_like(self.dynamics_W)
    
    for i in range(self.dynamics_W.shape[0]):
        for j in range(self.dynamics_W.shape[1]):
            self.dynamics_W[i, j] += eps
            pred_plus = self.decode(
                self.predict_next_state(state, action)
            )
            loss_plus = np.mean((pred_plus - next_obs) ** 2)
            
            self.dynamics_W[i, j] -= 2*eps
            pred_minus = self.decode(
                self.predict_next_state(state, action)
            )
            loss_minus = np.mean((pred_minus - next_obs) ** 2)
            
            self.dynamics_W[i, j] += eps  # 恢复
            grad_W[i, j] = (loss_plus - loss_minus) / (2*eps)
    
    # 更新
    self.dynamics_W -= self.lr * grad_W
    
    self.loss_history.append(loss)
    return loss
```

模块7：自我意识模块——AI的“元认知”
核心思想：真正的智能不仅需要感知和行动，还需要知道自己知道什么、不知道什么——这就是元认知（自我意识的雏形）。在NIEA中，我们通过元学习器来实现：一个高阶神经网络，监控低阶模块的状态，评估不确定性，并调整学习策略。

生物对应：前额叶皮层（尤其是背外侧前额叶）负责元认知——我们能反思自己的思维过程，评估自己的知识边界，并决定何时需要更多信息。

原理：不确定性感知与自适应学习
自我意识模块的核心功能：

置信度评估：对每个预测输出一个不确定性估计

学习率自适应：根据不确定性动态调整学习速度

注意力分配：决定哪些输入值得更多“思考”

元记忆：知道自己记住了什么、忘记了什么

python
"""
模块7：自我意识模块（Self-Awareness / Metacognition）
演示AI如何评估自己的知识边界并自适应学习
"""

import numpy as np
import matplotlib.pyplot as plt
from collections import deque

class MetacognitiveModule:
"""元认知模块：监控和调节底层学习过程"""

```
def __init__(self, input_dim=10, hidden_dim=16, memory_size=100):
    """
    Args:
        input_dim: 输入特征维度（来自底层模块的状态）
        hidden_dim: 隐藏层维度
        memory_size: 经验记忆容量
    """
    self.input_dim = input_dim
    self.hidden_dim = hidden_dim
    
    # 元学习网络：评估置信度和学习率调节因子
    self.W1 = np.random.randn(hidden_dim, input_dim) * 0.1
    self.b1 = np.zeros(hidden_dim)
    self.W2 = np.random.randn(2, hidden_dim) * 0.1  # 输出：[置信度, 学习率因子]
    self.b2 = np.zeros(2)
    
    # 经验记忆（用于元学习训练）
    self.memory = deque(maxlen=memory_size)
    
    # 历史记录
    self.confidence_history = []
    self.actual_error_history = []
    self.calibration_history = []

def forward(self, state_vector):
    """
    前向传播：根据当前状态评估置信度和建议学习率
    
    Args:
        state_vector: 底层模块的状态表示（如信念分布、预测误差等）
    
    Returns:
        confidence: 置信度 (0-1)
        lr_factor: 学习率调节因子 (>0)
    """
    # 隐藏层
    h = np.tanh(self.W1 @ state_vector + self.b1)
    
    # 输出层
    output = self.W2 @ h + self.b2
    
    # Sigmoid 激活
    confidence = 1.0 / (1.0 + np.exp(-output[0]))
    lr_factor = np.exp(output[1])  # 确保正值
    
    return confidence, lr_factor

def evaluate_knowledge_boundary(self, state_vector, task_difficulty):
    """
    评估知识边界：知道什么 vs 不知道什么
    
    Returns:
        knows: 是否认为自己知道答案
        should_explore: 是否应该探索而不是利用
    """
    confidence, _ = self.forward(state_vector)
    
    # 根据任务难度调整阈值
    threshold = 0.5 + 0.2 * task_difficulty
    
    knows = confidence > threshold
    should_explore = confidence < 0.7  # 低于70%置信度时倾向探索
    
    return knows, should_explore, confidence

def update_meta_knowledge(self, state_vector, predicted_confidence, 
                          actual_outcome, learning_outcome):
    """
    更新元知识：根据实际表现校准自我评估能力
    
    Args:
        state_vector: 当时的状态
        predicted_confidence: 预测的置信度
        actual_outcome: 实际结果（是否正确）
        learning_outcome: 学习效果（误差下降量）
    """
    # 存储经验
    self.memory.append({
        'state': state_vector.copy(),
        'pred_conf': predicted_confidence,
        'actual': actual_outcome,
        'learning': learning_outcome
    })
    
    # 校准误差：预测置信度 vs 实际准确率
    calibration_error = predicted_confidence - actual_outcome
    
    # 简单元学习更新（基于最近记忆）
    if len(self.memory) >= 10:
        self._meta_learning_step()
    
    self.calibration_history.append(calibration_error)

def _meta_learning_step(self, lr=0.01):
    """元学习：改进置信度校准能力"""
    # 采样最近的记忆
    batch = list(self.memory)[-10:]
    
    for exp in batch:
        state = exp['state']
        pred_conf = exp['pred_conf']
        actual = exp['actual']
        
        # 前向
        conf, _ = self.forward(state)
        
        # 误差
        error = conf - actual
        
        # 反向传播（简化版）
        h = np.tanh(self.W1 @ state + self.b1)
        dh = (1 - h**2)
        
        # 输出层梯度
        d_output = error * conf * (1 - conf)
        
        # 更新
        self.W2 -= lr * np.outer(d_output, h)
        self.b2 -= lr * d_output
        
        # 隐藏层梯度
        d_hidden = (self.W2[0] * d_output) * dh
        self.W1 -= lr * np.outer(d_hidden, state)
        self.b1 -= lr * d_hidden

def suggest_attention(self, input_features):
    """
    建议注意力分配：哪些特征值得更多关注
    """
    # 基于特征的不确定性
    attention_weights = np.abs(input_features) / (np.sum(np.abs(input_features)) + 1e-10)
    return attention_weights

def get_metacognitive_state(self):
    """获取元认知状态摘要"""
    if len(self.calibration_history) > 0:
        avg_calibration = np.mean(np.abs(self.calibration_history[-50:]))
        calibration_quality = 1.0 / (1.0 + avg_calibration)
    else:
        calibration_quality = 0.5
    
    return {
        'calibration_quality': calibration_quality,
        'memory_size': len(self.memory),
        'recent_confidence': np.mean(self.confidence_history[-20:]) if self.confidence_history else 0.5
    }
```

"""
NIEA完整训练流程：从婴儿到博士
将6个模块整合成端到端的训练循环
"""

import numpy as np
import matplotlib.pyplot as plt
from collections import deque
import time

# ========== 整合所有模块 ==========

class NIEABrain:
"""
神经同构自进化架构（NIEA）—— 完整大脑模型

```
整合：
- 模块1: LIF脉冲神经元（计算基元）
- 模块2: STDP学习（突触可塑性）
- 模块3: 预测编码（世界模型）
- 模块4: 主动推理（好奇心驱动）
- 模块5: HDC记忆（知识存储）
- 模块6: 元认知（自我意识）
"""

def __init__(self, 
             input_dim=10, 
             hidden_dim=64, 
             action_dim=4,
             hd_dim=1000,
             development_stages=True):
    """
    Args:
        input_dim: 感知输入维度
        hidden_dim: 隐藏层维度
        action_dim: 行动空间维度
        hd_dim: HDC记忆维度
        development_stages: 是否模拟发展阶段性
    """
    # 架构参数
    self.input_dim = input_dim
    self.hidden_dim = hidden_dim
    self.action_dim = action_dim
    
    # 发展阶段
    self.development_stages = development_stages
    self.age = 0  # 以经验步数衡量
    self.stage = 'infant'  # infant -> child -> adolescent -> adult -> expert
    
    # ===== 模块1: 脉冲神经元层 =====
    self.snn_layer = LIFLayer(input_dim, hidden_dim)
    
    # ===== 模块2: STDP学习 =====
    self.stdp_layer = STDPLayer(hidden_dim, hidden_dim)
    
    # ===== 模块3: 预测编码网络 =====
    self.pcn = SimplePCN(hidden_dim, 32, action_dim)
    
    # ===== 模块4: 主动推理引擎 =====
    self.active_inference = CuriosityEngine(action_dim)
    
    # ===== 模块5: HDC记忆系统 =====
    self.hd_memory = HDCMemory(hd_dim)
    
    # ===== 模块6: 元认知模块 =====
    self.metacognition = SimpleMetacognition()
    
    # 经验回放（用于巩固学习）
    self.experience_buffer = deque(maxlen=1000)
    
    # 统计指标
    self.stats = {
        'prediction_errors': [],
        'curiosity_levels': [],
        'knowledge_growth': [],
        'stage_transitions': []
    }

def perceive(self, sensory_input):
    """
    感知：将原始输入转化为内部表示
    
    流程：
    1. 脉冲编码（模拟视网膜/耳蜗）
    2. 特征提取（SNN前向传播）
    3. 预测生成（PCN预测下一个状态）
    """
    # 脉冲编码
    spike_train = self._encode_to_spikes(sensory_input)
    
    # SNN处理
    hidden_spikes, _ = self.snn_layer.forward(spike_train)
    hidden_state = np.mean(hidden_spikes, axis=0)
    
    # PCN预测
    prediction, prediction_error = self.pcn.predict(hidden_state)
    
    return hidden_state, prediction, prediction_error

def think(self, hidden_state, external_reward=0.0):
    """
    思考：内部认知过程
    
    流程：
    1. 元认知评估（我有多确定？）
    2. 好奇心计算（应该探索什么？）
    3. 行动选择（主动推理）
    4. 记忆检索（相关经验）
    """
    # 元认知评估
    confidence = self.metacognition.evaluate(hidden_state)
    
    # 计算好奇心（内在动机）
    curiosity = self.active_inference.compute_curiosity(
        hidden_state, self.pcn.get_prediction_uncertainty()
    )
    
    # 选择行动
    action_probs = self.active_inference.select_action(
        hidden_state, curiosity, external_reward
    )
    action = np.random.choice(self.action_dim, p=action_probs)
    
    # 从HDC记忆中检索相关经验
    memory_context = self.hd_memory.query(hidden_state, top_k=3)
    
    return {
        'action': action,
        'confidence': confidence,
        'curiosity': curiosity,
        'memory_context': memory_context
    }

def learn(self, hidden_state, action, next_hidden_state, 
          reward, prediction_error):
    """
    学习：更新所有可塑性组件
    
    流程：
    1. STDP无监督学习（基于脉冲时序）
    2. PCN预测学习（最小化预测误差）
    3. HDC记忆巩固（存储重要经验）
    4. 元认知校准（改进自我评估）
    """
    # STDP学习（局部、在线）
    stdp_update = self.stdp_layer.update(hidden_state, next_hidden_state)
    
    # PCN学习（全局、但生物可行）
    pcn_loss = self.pcn.learn(hidden_state, action, next_hidden_state)
    
    # 决定是否存储到长期记忆
    surprise = np.linalg.norm(prediction_error)
    if surprise > 0.5 or reward > 0:
        self.hd_memory.store(hidden_state, {
            'action': action,
            'outcome': next_hidden_state,
            'reward': reward,
            'surprise': surprise
        })
    
    # 元认知校准
    self.metacognition.calibrate(hidden_state, prediction_error)
    
    # 经验回放（睡眠时的记忆巩固）
    self.experience_buffer.append({
        'state': hidden_state,
        'action': action,
        'next_state': next_hidden_state,
        'reward': reward
    })
    
    # 定期巩固
    if len(self.experience_buffer) >= 100 and np.random.random() < 0.1:
        self._consolidate_memory()
    
    return {
        'stdp_update': stdp_update,
        'pcn_loss': pcn_loss,
        'surprise': surprise
    }

def _encode_to_spikes(self, sensory_input):
    """将连续值编码为脉冲序列（速率编码）"""
    n_steps = 20
    spike_train = np.zeros((n_steps, self.input_dim))
    
    for t in range(n_steps):
        # 泊松脉冲生成
        rates = np.clip(sensory_input, 0, 1) * 50  # Hz
        probs = rates / 1000  # 每毫秒概率
        spike_train[t] = (np.random.random(self.input_dim) < probs).astype(float)
    
    return spike_train

def _consolidate_memory(self):
    """记忆巩固（模拟睡眠）"""
    if len(self.experience_buffer) < 10:
        return
    
    # 采样经验
    batch = list(self.experience_buffer)[-20:]
    
    # 重放学习
    for exp in batch:
        # PCN通过重放巩固
        self.pcn.replay_learn(exp['state'], exp['action'], exp['next_state'])

def develop(self):
    """发展阶段评估与转换"""
    self.age += 1
    
    # 根据经验量评估发展阶段
    if self.development_stages:
        if self.age < 500:
            self.stage = 'infant'
        elif self.age < 2000:
            self.stage = 'child'
        elif self.age < 5000:
            self.stage = 'adolescent'
        elif self.age < 10000:
            self.stage = 'adult'
        else:
            self.stage = 'expert'
    
    # 记录阶段转换
    if self.age in [500, 2000, 5000, 10000]:
        self.stats['stage_transitions'].append((self.age, self.stage))
        print(f"\n发展阶段转换: {self.stage.upper()} (经验: {self.age}步)")

def get_state_summary(self):
    """获取当前状态摘要"""
    return {
        'age': self.age,
        'stage': self.stage,
        'knowledge_vectors': len(self.hd_memory.memory),
        'avg_prediction_error': np.mean(self.stats['prediction_errors'][-100:]) if self.stats['prediction_errors'] else 0,
        'curiosity': self.active_inference.curiosity_level,
        'confidence_calibration': self.metacognition.calibration_score
    }

class LIFLayer:
"""简化的LIF脉冲神经元层"""
def __init__(self, input\_dim, hidden\_dim):
self.W = np.random.randn(hidden\_dim, input\_dim) \* 0.1
self.v = np.zeros(hidden\_dim)
self.threshold = 1.0
self.tau = 10.0

```
def forward(self, spike_train):
    n_steps = spike_train.shape[0]
    spikes = np.zeros((n_steps, len(self.v)))
    
    for t in range(n_steps):
        current = self.W @ spike_train[t]
        self.v += (-self.v + current) / self.tau
        spikes[t] = (self.v >= self.threshold).astype(float)
        self.v[spikes[t] == 1] = 0
    
    return spikes, self.v
```

class STDPLayer:
"""简化的STDP学习层"""
def __init__(self, input\_dim, output\_dim):
self.W = np.random.randn(output\_dim, input\_dim) \* 0.1
self.A\_plus = 0.01
self.A\_minus = 0.012
self.trace\_pre = np.zeros(input\_dim)
self.trace\_post = np.zeros(output\_dim)
self.tau\_trace = 20.0

```
def update(self, pre_act, post_act):
    # 更新迹
    self.trace_pre = self.trace_pre * 0.9 + pre_act
    self.trace_post = self.trace_post * 0.9 + post_act
    
    # STDP更新
    dw = (self.A_plus * np.outer(post_act, self.trace_pre) - 
          self.A_minus * np.outer(self.trace_post, pre_act))
    self.W += dw * 0.01
    
    return np.mean(np.abs(dw))
```

class SimplePCN:
"""简化的预测编码网络"""
def __init__(self, state\_dim, hidden\_dim, action\_dim):
self.W\_pred = np.random.randn(state\_dim, state\_dim + action\_dim) \* 0.1
self.prediction = np.zeros(state\_dim)
self.uncertainty = 1.0

```
def predict(self, state):
    self.prediction = state.copy()  # 简化
    return self.prediction, state - self.prediction

def learn(self, state, action, next_state):
    sa = np.concatenate([state, [action]])
    pred = self.W_pred @ sa
    error = next_state - pred
    self.W_pred += 0.01 * np.outer(error, sa)
    self.uncertainty = np.mean(error**2)
    return self.uncertainty

def get_prediction_uncertainty(self):
    return self.uncertainty

def replay_learn(self, state, action, next_state):
    self.learn(state, action, next_state)
```

class CuriosityEngine:
"""主动推理 / 好奇心引擎"""
def __init__(self, action\_dim):
self.action\_dim = action\_dim
self.curiosity\_level = 1.0
self.exploration\_bonus = np.ones(action\_dim)

```
def compute_curiosity(self, state, uncertainty):
    self.curiosity_level = uncertainty
    return self.curiosity_level

def select_action(self, state, curiosity, reward):
    # 探索 vs 利用权衡
    probs = np.ones(self.action_dim) / self.action_dim
    probs += self.exploration_bonus * curiosity * 0.1
    probs += np.random.random(self.action_dim) * 0.05
    return probs / probs.sum()
```

class HDCMemory:
"""超维计算记忆系统"""
def __init__(self, dim=1000):

self.counter = 0

def store(self, key_vector, value):

    self.memory[key_hash] = value
    self.counter += 1

def query(self, query_vector, top_k=3):
    # 简化：返回最近存储的
    return list(self.memory.values())[-top_k:] if self.memory else []

def _vector_to_key(self, vec):

```

def evaluate(self, state):
    # 基于状态熵估计置信度
    entropy = -np.sum(np.abs(state) * np.log(np.abs(state) + 1e-10))
    confidence = 1.0 / (1.0 + entropy)
    self.confidence_history.append(confidence)
    return confidence

def calibrate(self, state, prediction_error):
    error_norm = np.linalg.norm(prediction_error)
    self.error_history.append(error_norm)
    
    if len(self.error_history) > 20:
        recent_conf = np.mean(self.confidence_history[-20:])
        recent_err = np.mean(self.error_history[-20:])
        self.calibration_score = 1.0 - min(1.0, abs(recent_conf - (1.0 - recent_err)))

    """执行行动，返回新状态和奖励"""

    action_vec = np.zeros(4)

工业级实现勘误与改进说明

本文档中的代码为概念验证级别的简化实现，用于阐述核心原理。在将其转化为工业级Python库(niea)的过程中，发现并修复了以下关键问题:

1. STDP模块: 资格迹替代最近脉冲时间 -- 仅记录最近脉冲时间无法正确处理多脉冲时间积分，工业级实现使用指数衰减的资格迹机制。

2. PCN模块: 梯度符号错误修复 -- 推理步骤和学习步骤的梯度符号均反转，导致自由能上升而非下降。修复后自由能正确收敛。

3. HDC模块: store_association绑定bug -- 原代码使用bind(key, key)绑定键与自身，正确应为bind(key, value)。

4. HDC模块: 序列编码位置向量不一致 -- store_episode和query_episode使用不同位置向量，修复后统一使用_get_position_vector()。

5. 世界模型: 数值梯度替换为解析梯度 -- 逐元素扰动法时间复杂度O(n_params)，解析反向传播为O(1)，速度提升1000倍以上。

6. LIF神经元: 返回类型修复 -- step方法返回int改为返回bool。

7. 全局: 梯度裁剪 -- 所有学习模块添加梯度范数裁剪，防止数值爆炸。

8. 全局: 参数验证 -- 所有模块添加完整参数验证(正值、维度、范围检查)。

9. 集成模块: 简化子模块替换为工业级子模块 -- LIFLayer/SimplePCN/CuriosityEngine等替换为ALIFNeuron/PredictiveCodingNetwork/ActiveInferenceAgent。

10. 主动推理: 完整变分推断 -- 使用互信息计算认知价值，Dirichlet先验进行贝叶斯后验更新。

11. World Model: 死代码enc_W2/enc_b2 -- 初始化了enc_W2和enc_b2但从未使用(encoder走的是enc_W_mu/enc_W_logvar路径)，已清理。

12. World Model: 重参数化梯度不完整 -- d_logvar缺少std*eps项的重参数化梯度，导致VAE训练时方差梯度计算错误。修复为: d_logvar = d_z_total * 0.5 * (z - mu) / (std + eps)。

13. Brain: 维度截断丢弃信息 -- 原代码使用hidden_state[:input_dim]截断，当hidden_dim > input_dim时丢弃大量信息。修复: 使用可学习的投影矩阵 state_to_obs_W 将hidden_dim投影到input_dim。

14. Brain: STDP与SNN未连接 -- STDP层独立于SNN层运行，STDP学到的权重没有影响SNN的输出。修复: STDP作为hidden-to-hidden循环连接，其输出作为循环电流注入SNN层。

15. Brain: 世界模型只训练不使用 -- 训练了世界模型但think步骤中没有用它做规划。修复: 添加_imagine_action_outcomes方法，对每个动作想象结果。

16. Brain: PCN predict_next维度可能不匹配 -- predict_next返回底层维度预测，但与hidden_state做差时维度可能不匹配。修复: 根据PCN层大小选择正确的预测来源。

17. World Model: 重参数化梯度修复(第二次) -- 原修复仍不正确。正确的重参数化梯度: d_logvar = d_z * 0.5 * eps * std, 其中eps是采样噪声, std = exp(0.5*logvar)。之前的修复使用了(z-mu)/std，这在数学上等价但不稳定(当std接近0时除零)。

18. Brain: HDC记忆检索键不匹配 -- _query_memory使用hash生成key，_store_to_memory使用age生成key，导致永远查不到存储的记忆。修复: _query_memory使用状态向量直接检索，_store_to_memory使用有意义的HDC向量作为概念向量。

19. Brain: SNN神经元参数导致零输出 -- 使用生物真实参数(v_rest=-65, v_threshold=-50)但输入电流不足，导致SNN层输出全零，整个感知链路瘫痪。修复: 调整为计算友好参数(v_rest=0, v_threshold=0.5)，确保SNN层能正常产生脉冲。

20. Brain: 脉冲编码输入信号太弱 -- _encode_to_spikes使用clip(input, 0, 1)但环境观测值可能为负或大于1，导致信号丢失。修复: 添加归一化(shift+scale)和自适应脉冲率。

理论补充: 全局工作空间意识涌现

全局工作空间理论(Global Workspace Theory, Baars 1988)认为意识是一个"全局广播"系统:

1. 模块竞争: 每个专门化模块产生"出价"(激活水平)
2. 胜者选择: 最高出价超过竞争阈值的模块获得工作空间访问权
3. 全局广播: 胜者将其输出广播给所有其他模块
4. 意识点火(Ignition): 当多个模块协同激活时，发生相变(意识"点火")
5. 动态阈值: 访问阈值根据近期活动自适应调整

意识级别估计 = 0.4 * 点火频率 + 0.3 * 工作空间激活水平 + 0.3 * 胜者模块多样性

代码实现: niea.consciousness.GlobalWorkspace

理论补充: 结构自进化

NIEA的结构自进化机制在比学习更慢的时间尺度上运行:

1. 突触修剪(Pruning): 移除权重幅度低于阈值的连接，提高效率
2. 神经发生(Neurogenesis): 当性能平台化时，添加新神经元扩展容量
3. 结构可塑性(Rewiring): 基于活动相关性重新布线连接
4. 性能平台检测: 检测学习是否停滞，触发结构性改变
5. 周期性进化: 每N步训练后执行一次进化循环

代码实现: niea.evolution.StructuralEvolution

理论补充: 互补记忆系统

互补学习系统理论(Complementary Learning Systems, McClelland et al. 1995):

海马体存储(Hippocampal Store):
- 快速一次性编码
- 模式分离(减少干扰)
- 有限容量、短期保留
- 驱动记忆巩固

皮层存储(Cortical Store):
- 慢速渐进整合
- 模式完成(泛化)
- 大容量、长期稳定
- 语义知识提取

巩固过程: 在"睡眠"期间，将记忆从海马体转移到皮层存储

代码实现: niea.memory.consolidation.ComplementaryMemorySystem

57. PCN梯度裁剪扭曲方向 -- 对dW和db独立裁剪，当两者量级差异大时，裁剪后的梯度方向偏离原始方向。修复: 联合裁剪，将dW和db展平后计算单一范数，统一缩放。

第十轮审查修复(训练正确性与GPU一致性):

58. perceive(next_obs)覆盖PCN保存状态(致命) -- 训练循环perceive(obs)->perceive(next_obs)->learn()中，第二次perceive覆盖了第一次保存的PCN预测误差，导致PCN从next_obs的误差学习而非obs的。修复: 保存完整PCN状态(含层状态)到_saved_pcn_state，仅在未被保存时才保存(防止覆盖)，learn()消费后清除。

59. _consolidate_memory使用过期PCN误差 -- 记忆巩固时使用_saved_pcn_prediction_errors(来自最近一次perceive)，而非回放经验对应的误差。修复: 巩固时对回放状态运行pcn.infer(state)获取新鲜预测误差，再调用pcn.learn()。

60. GPU前向不处理ALIF自适应阈值(致命) -- GPU路径忽略ALIF神经元的自适应变量w，导致使用默认ALIF神经元时GPU结果完全错误。修复: GPU路径添加w_adapt变量，每步衰减w，spike时w+=1.0，有效阈值=v_threshold+beta*w。

61. GPU前向不维护神经元状态(致命) -- GPU路径每次从静息态(v=v_rest, refractory=0)开始，忽略当前神经元状态，导致连续forward调用间状态不连续。修复: 从_sync_from_neurons()获取当前状态初始化GPU张量。

62. _sync_from/to_neurons引用不存在属性 -- 使用neuron.refractory_counter和neuron.total_current，但LIFNeuron实际属性为refractory_timer和i_syn。修复: 修正属性名。

63. GPU不应期不重置膜电位(致命) -- CPU在不应期设置v=v_reset和i_syn=0.0，GPU路径在不应期保持v不变且电流正常注入，导致GPU和CPU产生完全不同的脉冲模式。修复: GPU路径在不应期设置v=v_reset，电流置零。

64. GPU不实现突触滤波 -- CPU当tau_s设置时使用一阶低通滤波i_syn = current + (i_syn - current) * exp(-dt/tau_s)，GPU直接使用current。修复: GPU路径添加tau_s支持。

65. GPU不应期计时器可变负 -- CPU使用max(0, timer-dt)防止计时器变负，GPU使用timer-dt可无限变负。修复: 使用torch.clamp(refractory-dt, min=0.0)。

66. _select_action_from_imagination使用错误可取性度量 -- 使用np.mean(np.abs(obs_pred))作为动作可取性，这只是观测的平均幅度，与奖励无关。修复: 使用世界模型的predict_reward(z_next)作为可取性度量。

67. WorldModel train_step不使用GPU -- 即使设置了device="cuda"，train_step仍全部使用NumPy计算。修复: 添加_train_step_gpu方法，将前向传播和梯度计算移到PyTorch GPU上，权重更新时转回NumPy。

第十一轮审查修复(梯度一致性与checkpoint完整性):

68. PCNLayer非线性分支梯度裁剪不一致 -- 线性分支使用联合裁剪(dW+db展平后计算单一范数)，非线性分支只对dW独立裁剪且完全跳过db更新。修复: 非线性分支也使用联合裁剪并更新bias。

69. Trainer checkpoint恢复缺失三个新模块 -- _reconstruct_brain_state只恢复7个模块，缺少global_workspace、structural_evolution、complementary_memory。从checkpoint加载后这三个模块的状态丢失。修复: 添加到模块列表。

70. Trainer _flatten_dict不处理bool类型 -- 布尔值(如done标志)无法被np.savez正确序列化。修复: isinstance检查添加bool。

71. Trainer checkpoint恢复缺失关键字段 -- consolidation_counter和perception_projection_W/input_dim未从checkpoint恢复。修复: 添加恢复逻辑。

第十二轮审查修复(梯度正确性与训练信号):

72. PCNLayer非线性分支梯度计算错误 -- 非线性分支(f=tanh)的权重梯度应使用链式法则: dW = sigma * outer(error_below * (1 - f^2), x)，而非简单的 dW = sigma * outer(error_below, x)。bias梯度同理: db = sigma * error_below * (1 - f^2)。原代码忽略了tanh的导数项，导致非线性PCN权重更新方向错误。修复: 正确计算tanh导数。

73. evolve_module在平台化时调用grow_neurons导致维度不匹配 -- brain._evolve_structure调用evolve_module，但evolve_module内部在检测到性能平台化时会调用grow_neurons添加新神经元，改变权重矩阵维度，导致所有下游模块崩溃。修复: _evolve_structure直接调用prune_weights而非evolve_module。

74. Trainer训练循环未传递上一步环境奖励给think() -- think()的external_reward始终为0.0，主动推理无法利用奖励信号调整偏好和策略。修复: 保存上一步的env_reward，在下一步think()时传入。

第十三轮审查修复(非线性PCN偏置遗漏):

75. 非线性PCN预测函数忽略偏置项 -- _tanh_predict(x, W)返回tanh(W@x)，完全忽略了偏置b。PCNLayer.predict()在非线性模式下调用f(x, W)而非f(x, W, b)，导致偏置永远不参与预测计算。修复: _tanh_predict改为_tanh_predict(x, W, b)返回tanh(W@x+b)，predict()传入b参数。

76. 非线性PCN导数函数忽略偏置项 -- _tanh_predict_deriv(x, W)计算W*(1-tanh(W@x)^2)，但应该使用W@x+b而非W@x。修复: _tanh_predict_deriv添加b参数。

77. PCNLayer.compute_prediction_deriv不传递偏置 -- 调用f_deriv(x, self.W)而不传self.b。修复: 传入self.b。

78. PCNLayer.learning_step非线性分支计算预测时不传偏置 -- prediction = self.f(x, self.W)忽略了b，导致梯度计算中的f_deriv_elem基于错误的预测值。修复: 改为self.f(x, self.W, self.b)。

第十四轮审查修复(稀疏架构与人脑规模参数):

79. 密集矩阵无法支持人脑规模 -- 原实现使用密集numpy数组存储突触权重，当hidden_dim=860亿时，单个W矩阵需要5.9ZB内存（远超全球存储总量）。人脑之所以可行是因为它是稀疏的——每个神经元平均只连接~7000个突触（而非860亿）。修复: 创建niea.neurons.sparse_connectivity模块，实现SparseConnectivity类（CSR格式稀疏权重矩阵管理器）和BrainScaleConfig类（人脑规模配置）。

80. SNN层不支持稀疏连接 -- SNNLayer使用密集W矩阵，无法扩展到人脑规模。修复: 添加sparse_connectivity和synapses_per_neuron参数，当sparse=True时使用scipy.sparse.csr_matrix替代密集numpy数组，矩阵乘法改为W.dot(x)兼容两种格式。

81. STDP层不支持稀疏突触 -- STDPLayer使用密集W矩阵，STDP更新遍历所有突触（包括不存在的连接）。修复: 添加sparse_connectivity和synapses_per_neuron参数，稀疏模式下仅更新已有连接的权重，通过COO格式的row/col索引向量计算LTP/LTD。

82. PCN不支持稀疏预测编码 -- PCNLayer使用密集W矩阵，无法扩展到人脑规模的感觉皮层预测通路。修复: 添加sparse和synapses_per_neuron参数，稀疏模式下使用scipy.sparse.csr_matrix，predict/inference_step/learning_step均支持稀疏矩阵运算，learning_step仅更新已有非零位置的权重。

83. Brain不支持人脑规模参数 -- NIEABrain默认参数远小于人脑规模（hidden_dim=256 vs 860亿）。修复: 添加brain_scale=True参数，启用时自动覆盖所有参数为人脑规模（hidden_dim=160亿、hd_dim=1500万、pcn_hidden_dim=80亿等），匹配人脑各区域神经元数量。添加sparse=True参数，启用稀疏连接（每神经元~7000突触）。

84. 训练脚本参数不合理 -- 之前将observation_dim设为860亿，但环境观测维度是压缩后的感知信号，不需要等于神经元数量。修复: 恢复observation_dim=128，通过brain_scale=True让SNN层自动扩展到人脑规模，感知信号从128维扩展到160亿维通过稀疏SNN层实现。
